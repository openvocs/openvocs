Import the following css files

@import "/css/ov_basic.css";

To use the dark color schema add the class dark to the body or a div sourounding your HTML code. To use the light color schema add the class light.
If the interface is indeted for touch interaction add the class touch to the body or a div sourounding your HTML code.
