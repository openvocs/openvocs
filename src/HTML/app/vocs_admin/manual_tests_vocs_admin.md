## Offline mode

* [ ] open existing project
* [ ] save project is enabled 
* [ ] trigger offline mode (e.g. disable server)
* [ ] save project is disabled
* [ ] dismiss offline mode (e.g. restart server)
* [ ] save project is enabled

## Offline mode (new project)

* [ ] open new project
* [ ] save project is disabled 
* [ ] trigger offline mode (e.g. disable server)
* [ ] save project is disabled
* [ ] dismiss offline mode (e.g. restart server)
* [ ] save project is disabled
